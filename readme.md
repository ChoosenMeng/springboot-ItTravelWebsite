# IT旅游网 项目介绍

一款常见的旅游网站，项目分为管理员端和用户端。管理员端（后台）主要进行旅游产品的维护，用户端（前台）主要进行旅游产品的展示。

---

+ ## 主要功能
	+ 管理员端
		+ 管理员管理
		+ 角色管理
		+ 权限管理
		+ 认证和授权
		+ 产品类型管理
		+ 旅游产品管理
	+ 用户端
		+ 用户注册
		+ 用户登录
		+ 查询旅游产品列表
		+ 查询旅游产品详情
		+ 收藏旅游产品
		+ 查询我的收藏

---

+ ## 技术选型
  + Java版本：JDK17
  + 数据库：MySQL8.0.22+Navicat
  + 后端框架：SpringBoot3+SpringMVC+Mybatis-Plus3.5.0
  + 权限控制：SpringSecurity
  + 前端框架：AdminLTE2
  + 模板引擎：Thymeleaf
  + 工具类：发邮件工具类、生成验证码工具类
  + 其他技术：lombok、ajax、logback
